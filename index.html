Config:

Instagram Video Downloader configuration is easy. You can change site title, description, Google Anaylitcs tracking id and ads with ease.

SITE URL:

Insert full http path to Instagram Downloader and make sure it ends with slash.

define('SITE_URL', 'http://yoursite.com/');
For example If you would like to edit the 728x90 ad, you would edit the following line from config.php file:

Ads Setup:

  /*Responsive for both desktop and mobile
        ad size 728x90
    */
    "ad728" => '<img src="https://dummyimage.com/728x90/343a40/fff" class="img-fluid">',
 

You would do the same if you want to edit other ads.

You'll also be able to edit other site settings such as title, description, etc by editing config.php file. For example, Google Anaylitcs tracking id can be added by editing the below line:

/** Google Analytics id, should be in this format:
      UA-12345678-1
*/
    "ga" => "UA-12345678-1",

How to setup Proxies:

The downloader may work without proxies initially, but in case it stops working, it is necessary to integrate proxies. This is because IG bans the IP address once they detect too many downloading requests.

  1. Open your file manager and navigate to the root directory of your downloader..
  2. Edit the "proxies.txt" file inside the "proxies" folder.
  3. You may add multiple proxies to this file, but make sure each proxy is on a new line.
  4. Ensure that each proxy is in the following format: "http://username:password@host:port". Replace "username" and "password" with your proxy login credentials, and "host" and "port" with the appropriate information for your proxy.
  5. Save the changes to the "proxies.txt" file.

How to fix downloader errors:

If the downloader stops working, just check the log.txt file to see what's wrong. You'll find it in proxies/log.txt. If you're not sure what's happening, don't hesitate to ask us on Codester. Make sure to send us the log file so we can track the issue.

Usually, errors only pop up when you forget to add proxies to the downloader.
